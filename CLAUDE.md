# CLAUDE.md — mayaonkeys.com

## What this project is

A static website documenting Maya's piano learning journey — songs, milestones, videos, and a practice randomiser called "Oldie Goldie". Live at [mayaonkeys.com](https://www.mayaonkeys.com).

## Tech stack

- **Eleventy (11ty) v3** — static site generator
- **Nunjucks** — templating
- **Tailwind CSS v3** — styling
- **Decap CMS** — content management at `/admin`
- **Netlify** — hosting and deployment

## Running locally

```bash
npm install
npm start        # builds CSS then starts 11ty dev server at localhost:8080
npm run build    # production build to _site/
```

## Key files

| File | Purpose |
|---|---|
| `.eleventy.js` | Collections (songs, milestones, learnedYears), filters, passthrough copy |
| `_data/config.json` | Site-wide config — name, bio, start date, Instagram, fun facts |
| `_includes/base.njk` | Base layout — nav, footer, SEO meta tags (OG, Twitter Cards, canonical) |
| `_includes/song-card.njk` | Song card partial with external link auto-labelling |
| `_includes/milestone-entry.njk` | Milestone entry partial with image and external link |
| `admin/config.yml` | Decap CMS collection definitions and field schemas |
| `tailwind.config.js` | Brand colours, fonts, theme extensions |
| `netlify.toml` | Build command, publish dir, redirects |

## Content model

- **Songs** — `content/songs/*.md`. Front matter: `title`, `composer` (optional), `status` (`learned`/`learning`), `dateLearned`, `externalLink`, `externalLinkLabel`. Filenames should be date-prefixed (e.g. `2025-11-twinkle-twinkle.md`).
- **Milestones** — `content/milestones/*.md`. Front matter: `title`, `date`, `image`, `imageAlt`, `externalLink`, `externalLinkLabel`.
- **Site config** — `_data/config.json`.

Songs do not generate individual pages (`songs.11tydata.json` sets `permalink: false`). Milestones are list-only on `/milestones/`.

## Collections and filters (`.eleventy.js`)

- `collections.songs` — all songs sorted by `dateLearned` (newest first, currently-learning last)
- `collections.milestones` — all milestones sorted by date (newest first)
- `collections.learnedYears` — unique years extracted from learned songs (newest first)
- Filters: `dateDisplay`, `yearMonth`, `year`, `yearFromDate`, `limit`, `learnedSongs`, `learnedByYear`, `htmlDateString`

## External link auto-labelling

Songs and milestones support `externalLink`. The label is auto-detected from the URL domain:
- `youtube.com` / `youtu.be` → "Watch on YouTube"
- `instagram.com` → "View on Instagram"
- `tiktok.com` → "View on TikTok"
- `facebook.com` → "View on Facebook"
- Fallback → "View link"

Override with `externalLinkLabel`.

## Privacy rules

- No full name, school, or location anywhere on the site
- No analytics or tracking (yet)
- External links use `rel="noopener noreferrer"`
